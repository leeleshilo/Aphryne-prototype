# Aphryne Prototype
Portfolio project showcasing a luxury e-commerce website UI. Designed and developed using HTML, CSS, and Bootstrap with emphasis on responsive design, branding, and user experience.

## Overview
A luxury hair and accessories brand website prototype designed to showcase front-end development and UI/UX design skills. This project focuses on creating a premium, modern e-commerce landing experience with strong visual branding and a responsive layout.

## Disclaimer
This is a front-end demonstration project. Navigation and interactive elements are non-functional and are included for visual and layout purposes only.

## Purpose
To demonstrate:
- Luxury brand styling and visual storytelling
- Responsive web design principles
- Clean UI/UX layout for an e-commerce experience
- Ability to customize and transform a template into a unique design

## Design Approach
The design follows a minimalist luxury aesthetic, using refined typography, structured spacing, and a well-coordinated color palette. The layout guides users from brand messaging to product discovery, simulating a real online shopping experience.

## User Experience
The interface is structured to feel intuitive and engaging:
- Clear visuals
- Smooth content flow
- Section-based navigation feel
- Focus on brand immersion and product presentation

## Tools & Technologies
- HTML5
- CSS3
- Bootstrap 4 (layout and responsiveness)
- JavaScript (UI interactions: animations, navigation, carousel)
- Visual Studio Code

## Template Source
This project is based on a free Bootstrap template sourced from:
ThemeWagon (https://themewagon.com/)

The template was fully customized, including layout adjustments, styling changes, branding, and content restructuring to create a unique final product.

## Role
Lead Front-End Designer & Developer:
- Customized and restructured the original template
- Designed layout and visual hierarchy
- Implemented responsive behavior
- Refined styling for a luxury brand feel
- Applied semantic HTML structure

## Responsiveness
Built with a mobile-first approach using Bootstrap grid and utilities. The layout adapts across desktop, tablet, and mobile devices to ensure a consistent user experience.

## Performance Considerations
